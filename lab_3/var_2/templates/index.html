API Tutorial

  • /api/gtd/ - просмотр списка всех объектов (метод GET)

  • /api/gtd/{id} - просмотра одного объекта по ключевым значениям (метод GET)

    • /api/gtd?{params} - добавления нового объекта (метод POST)
    • /api/gtd/{id}?{params} - добавление нового объекта (метод POST)

  • /api/gtd/{id}?{params} - редактирования объекта (метод PUT)

  • /api/gtd/{id} - удаление объекта (метод DELETE)

  • /api/gtd/number/{month} - просмотр общего числа терактов в определённый месяц года (1..12) (метод GET)
