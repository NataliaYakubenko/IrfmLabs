API Tutorial

  • {{ api_prefix }}/ - просмотр списка всех объектов (метод GET)

  • {{ api_prefix }}/{id} - просмотра одного объекта по ключевым значениям (метод GET)

    • {{ api_prefix }}?{params} - добавления нового объекта (метод POST)
    • {{ api_prefix }}/{id}?{params} - добавление нового объекта (метод POST)

  • {{ api_prefix }}/{id}?{params} - редактирования объекта (метод PUT)

  • {{ api_prefix }}/{id} - удаление объекта (метод DELETE)

  • {{ api_prefix }}/number/{gender} - просмотр суммарного числа людей определённого пола, участвовавших в опросе (male, female) (метод GET)
